How to Setup SSH Keys on CloudLab?

Generate an SSH key

Run the following command in a terminal:

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
Generating public/private rsa key pair.

When prompted for a file name pick a name and enter it (LEAVE PASSWORD EMPTY WHEN ASKED):

Enter file in which to save the key (/Users/anask/.ssh/id_rsa): /Users/anask/.ssh/clab_rsa
Enter passphrase (empty for no passphrase):
Enter same passphrase again:


Note the locations of the generated keys:

Your identification has been saved in /Users/anask/.ssh/clab_rsa.
Your public key has been saved in /Users/anask/.ssh/clab_rsa.pub.

Copy your public key into another directory so that it becomes visible:

cp /Users/anask/.ssh/clab_rsa.pub ~/Desktop/

Upload the public key to CloudLab

Click on your username on the CloudLab navigation bar, then click on "Manage SSH Keys".


Then click on 'Load from file' to load the saved public key, which ends with '.pub'.

Next, click on 'Add key' to upload the key.
